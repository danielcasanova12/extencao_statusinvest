# Política de Privacidade – StatusInvest Latest Panel

A extensão **StatusInvest Latest Panel** não coleta, armazena ou compartilha informações pessoais dos usuários.

### Uso de permissões
- **Host (statusinvest.com.br)**: utilizado apenas para injetar o painel dentro do site StatusInvest.  
- **Storage**: usado exclusivamente para salvar configurações locais da extensão, como chave da API e preferências visuais.  
- **Código remoto (API do usuário)**: a extensão conecta-se somente à API configurada pelo próprio usuário para buscar dados exibidos no painel. Nenhum dado é enviado para terceiros.  

### Coleta de dados
Esta extensão **não coleta** dados de navegação, informações pessoais, nem compartilha dados com terceiros.
